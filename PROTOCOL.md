# بروتوكول الربط التقني (الـ Hub المشترك)

هذا المستند يوضح كيفية ربط (أيمن، محمود، سمسم) عبر مستودع (Git Repository) خاص.

## 1. الهيكلية
كل جهاز سيحتوي على مجلد `SHARED_HUB` متصل بنفس المستودع (Private Repo).

## 2. الملفات المتاحة للجميع
- `tasks.json`: المهام المطلوبة (من أيمن للآخرين).
- `results.json`: نتائج التنفيذ (من الآخرين لأيمن).
- `configs/`: ملفات الإعدادات المشتركة.
- `knowledge/`: ملفات المعرفة والمراجع (PDFs, Excel, etc).

## 3. خطوات التنفيذ لكل جهاز
1. **توليد مفتاح SSH:**
   ```bash
   ssh-keygen -t ed25519 -C "dr_ahmed_system"
   ```
2. **إضافة المفتاح لـ GitHub:** انسخ محتوى `~/.ssh/id_ed25519.pub` وضعه في إعدادات الـ Repo.
3. **الربط:**
   ```bash
   git clone git@github.com:USERNAME/SHARED_HUB.git
   ```

## 4. نظام العمل (الزتونة)
- **للقراءة:** `git pull`
- **للتنفيذ:** (يتم العمل محلياً)
- **للتسليم:** `git add . && git commit -m "update" && git push`

## 5. ملاحظة هامة للموارد
- المزامنة يدوية فقط (عند الحاجة) لضمان عدم استهلاك موارد السيرفر (Zero Cost).
